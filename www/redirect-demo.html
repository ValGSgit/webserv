üìñ About HTTP Redirects

301 Moved Permanently: The URL has permanently moved to a new location.

302 Found: Temporary redirect to another URL.

Location Header: Specifies the new URL for the resource.

Configuration Example:

    return 301 /;
}

location /github {
    return 301 https://github.com/ValGSgit/webserv;
}
