RFC 7231 Compliance

This server implements HTTP/1.1 methods according to RFC 7231 (Hypertext Transfer Protocol - Semantics and Content):

  • Safe methods (GET, HEAD, OPTIONS) do not modify resources
  • Idempotent methods (GET, HEAD, PUT, DELETE, OPTIONS) produce the same result when called multiple times
  • HEAD requests return identical headers to GET but without the body
  • OPTIONS requests return the Allow header listing permitted methods
  • Content-Length headers are preserved in HEAD responses
